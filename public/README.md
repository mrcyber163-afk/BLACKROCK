# BlackRock - Earn Money Platform

## 📱 About
BlackRock is a platform where users earn money by watching videos from TikTok, Facebook, YouTube, and Instagram.

## 🚀 Features
- User Registration with Zambian phone numbers
- Video tasks from multiple platforms
- Deposit system with Airtel/MTN Money
- Withdrawal to mobile money
- Referral system with bonuses
- Wealth Fund investment (180 days, 75% returns)
- Welcome bonus (100 ZMW)
- Admin dashboard
- Real-time notifications

## 🔧 Technologies
- HTML5, CSS3, JavaScript
- Firebase (Auth, Firestore, Realtime Database)
- Font Awesome Icons
- Netlify Hosting

## 📦 Deployment on Netlify
1. Push code to GitHub
2. Connect repository to Netlify
3. Deploy automatically

## 🔐 Environment Variables
No environment variables needed - Firebase config is included in the code.
